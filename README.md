# 미션 - 크리스마스 프로모션

## 기능 목록 📝 
1. 식당 소개 및 인사말 조회  
2. 예상 방문 날짜 입력  
    - **예외처리**: 숫자가 아닌 것을 입력할 경우  
    - **예외처리**: 숫자의 범위가 1~31을 벗어날 경우   
3. 메뉴와 개수 입력
    - **예외처리**: 입력 형식(하이픈, 대괄호, 쉼표)이 다를 경우  
    - **예외처리**: 메뉴판에 없는 메뉴 입력할 경유   
    - **예외처리**: 중복된 메뉴를 입력할 경우  
    - **예외처리**: 메뉴의 갯수가 숫자가 아닐 경우 
    - **예외처리**: 숫자가 자연수가 아닐 경우
    - **예외처리**: 입력한 메뉴의 수가 20개 넘을 경우  
    - **예외처리**: 입력한 메뉴 중에 음료만 주문할 경우    
4. 주문한 총 메뉴 조회  
5. 할인 전 총주문 금액 조회  
6. 증정 메뉴 조회
7. 혜택 내역 조회
8. 총혜택 금액 조회
9. 할인 후 예상 결제 금액 조회
10. 12월 이벤트 배지 조회

## 확장 가능성 고려 🆙  
1. 크리스마스 디데이 할인 방식 변경
